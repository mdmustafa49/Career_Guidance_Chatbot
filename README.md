# Career_Guidance_Chatbot
NLP Project - Career guidance chatbot using Flask


This is an NLP-based Career Guidance Chatbot developed using Python and Flask. It helps students choose the right career path by analyzing their queries.


--->  Features:
- Hybrid NLP (Rule-based + Machine Learning)
- TF-IDF + Logistic Regression
- 16 career domains covered
- Confidence score for each response
- Web-based chat interface


--->  Technologies Used:
- Python
- Flask
- NLTK
- Scikit-learn
- HTML, CSS, JavaScript


--->   How to Run

1. Install dependencies:
                pip install -r requirements.txt
2. Run the server:
                python backend/app.py


--->   Future Improvements
- Add BERT / Transformer models
- Add conversational memory
- Integrate LLM APIs


--->   Description
This project demonstrates the application of Natural Language Processing in solving real-world problems like career guidance.
